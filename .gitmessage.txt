# <-------------------- best practice -------------------->
# - コミットの種類
#   - Fix: バグ修正
#   - Hotfix: クリティカルなバグ修正
#   - Add: ファイル追加または機能追加
#   - Change: 仕様変更やロジック変更
#   - Clean: リファクタリングなどの整理
#   - Remove: ファイル削除または機能削除
#
# - タイトル
#   - タイトルを50字以内におさめる
#   - タイトルの後は1行空けて本文を書く
#   - タイトルの文頭を大文字にする
#   - タイトルの文末にピリオドを付けない
#   - タイトルは命令形で記述する
#
# - 本文 
#   - 本文は1行あたり72字以内におさめる
#   - 本文ではどのようにではなく何をとなぜを説明する
